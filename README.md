Super easy way to que up processes needing resources, or balls.

You hold a ball and put it back when you're done. If you're waiting for a ball, no one can take it.

How to use:
```
# How to use
GET /[user]/hold/[ball]
GET /[user]/put/[ball]
GET /[user]/wait_for/[ball]
```
